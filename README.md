# Ultimate Quiz App - Android

A premium, interactive Android application developed as part of the **Oasis Infobyte Internship**. This app challenges users with randomized general knowledge questions, featuring a modern Material Design interface and real-time performance tracking.

## ‚ú® Features
- **Randomized Gameplay:** Every session picks 10 random questions from a larger pool.
- **Real-time Feedback:** Options change to soft green (correct) or soft red (incorrect) to provide instant user feedback.
- **Progress Tracking:** A visual progress bar and question counter keep the user informed.
- **Clean UI:** Built with Material Design 3, featuring custom gradients and card-based layouts.
- **Score Persistence:** Uses Intent Extras to pass final scores to a dedicated Results screen.
